Github Starter

Steps

1. Make your own folder by using the tag "Md" (Make Directory).

2. Use the "CD" tag (Change Directory) to go into your newly created folder.

3.Use the "git init" tag to intialize your repository in Github.

4.Next you want to add a README.md file for you to be able to push your file into github.

5.Now commit the file and push it to github.
